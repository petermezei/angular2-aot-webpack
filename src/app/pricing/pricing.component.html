Getting started

Pricing

Our product is available for both private hosting and SaaS. You can easily download and install it on your machine, or purchase it as a monthly service.

For private hosting, you can download and install Slamby API and register it with an open-source license for free, and you can purchase a core-based license for commercial use.

For SaaS, you can select a required instance size and buy it with a monthly payment. The prices cover the standard Slamby license per core + hosting fee.

Core pricing

Standard Slamby API pricing, based on the processor core number. There is no query, usage, storage and memory limitation.
